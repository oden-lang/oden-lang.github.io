## Number Literals

The only supported number literal right now is that of the `int` type.

{% highlight go %}
123
-588
{% endhighlight %}

## Boolean Literals

There are two boolean literals, `true` and `false`, just as in Go.

{% highlight go %}
true
false
{% endhighlight %}

## String Literals

{% highlight go %}
"hello"
""
"\nomg\nnewlines"
{% endhighlight %}
